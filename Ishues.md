## 3. Flet UI Compatibility & UX

### Fixed: Flet Dropdown Crash
- **Issue**: `TypeError: Dropdown.__init__() got an unexpected keyword argument 'on_change'`. This occurred because certain Flet versions do not support the `on_change` kwarg during initialization.
- **Fix**: Refactored `graph_tab.py` and `x_ray_flet.py` to use `on_select` or post-init assignment, ensuring compatibility across desktop and web environments.

### UX Improvement: Center Start Action
- **Change**: Moved the **START SCAN** button from the left sidebar to the center of the landing page.
- **Reason**: Promotes a clearer "User Journey" and makes the primary action more prominent with a "Hero" style design and pulsing glow effect.

### Fixed: NameError SZ_SM
- **Issue**: `name 'SZ_SM' is not defined` in `graph_tab.py`.
- **Fix**: Resolved by importing missing layout constants from `UI.tabs.shared`.
